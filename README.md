# 项目简介
* three.js模拟太阳系运行。
* 具有行星自转、公转效果。
* 双击行星可拉近查看。
* 行星点击后会弹出相关文字描述。
# 相关技术
* vue2主体框架。
* three.js绘制3d图形。
# 项目使用
* 安装依赖：npm install
* 项目运行：npm run serve
* 项目打包：npm run build
# 详情说明
* 双击空白可暂停或开启行星自转、公转。
